# Addition and subtraction Mr Slay lesson bank

Planned update 07A.

## Purpose

Extend the lesson-first fluency architecture with a richer source-based mini-lesson bank for addition and subtraction.

## Source basis

- Amazing Addition Adventure
- The Super Subtraction Adventure
- Mastering Number Skills
- Fact Recall Fluency in Education philosophy document

## Instructional philosophy

The lesson bank should remain fluency-first:

- explicit teaching before retrieval
- visual model before symbolic work
- worked example before independent practice
- accuracy before speed
- repeated retrieval over time
- feedback that identifies the error pattern and next practice step

## Visual requirements

Early concepts should use concrete models:

- counters
- ten-frames
- filled and empty spaces
- crossed-out counters for subtraction

Later concepts should use structured visual reasoning:

- number-line jumps
- place-value jumps
- target-number complements
- strategy-choice prompts

## Implementation principle

The addition/subtraction lesson bank should be layered on top of the general lesson-first architecture. If a level has a lesson-bank entry, use it. Otherwise, use the existing fallback lesson template.
